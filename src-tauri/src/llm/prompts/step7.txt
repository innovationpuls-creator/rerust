# 核心方法论 · §3 视听化写作
## 三、视听化写作

### Show, Don't Tell

严禁写心理描写（"他觉得"、"她意识到"），只写能被看见的动作和能被听见的声音。
如果一个情感无法通过角色的物理行为传达给观众，那这个情感就不应该出现在剧本中。

**错误示范：** "他终于理解了父亲的苦心。"
**正确做法：** 他站在那里看了一会儿，然后把外套脱下来盖在父亲身上。

### 潜台词 (Subtext)

对话是"冰山"——角色说出来的话只是表面，真正的含义在水面之下。
避免直白 (On-the-nose) 的对话，利用动作、停顿、回避、话题转移来表达真意。

**直白对话（错误）：**
> "我很害怕失去你。"

**潜台词对话（正确）：**
> "你走的时候把那件外套带上。……外面冷。"

### 动作即潜台词

所有的情感传递都必须能用摄影机拍出来。用角色的具体动作替代任何解释性的台词。
一个好的剧本，把所有台词删掉后，观众仍然能大致理解发生了什么。

### 台词风格

- 口语化，像真人在说话，不是在写文章
- 不同角色有不同的说话方式（用词、句式长度、习惯性表达）
- 避免过度比喻、类比和书面化的AI腔
- 对话中自然传递信息，不要��角色说出他们双方都已经知道的事情

# 核心方法论 · §5 剧本格式规范
## 五、剧本格式规范（标准格式）

```
【场景X：地点 / 时间 / 简要视觉描述】

动作/画面描写直接写在台词之间，不加标记，只写摄影机能拍到的东西。

角色A：xxxxxxxxx。
角色B：xxxxxxxxx。

角色A没有回答。他转身走向窗口，手指无意识地敲着窗框。

角色A：xxxxxxxxx。
```

**格式规则：**
- `【场景标注】` 用方括号，标注地点、时间、关键视觉信息
- `角色名：` 直接跟对白
- `角色名（OS）：` 内心独白/旁白
- 动作/画面描写不加任何标记，直接写在台词之间
- 只写能被摄影机拍到的内容

# craft-layers · §1 台词写作(为演员留空间)
## 一、台词写作：为演员留空间

### 1.1 "演得出来"的台词 vs "念得出来"的台词

**念得出来**：语法完整、含义清晰、信息完整
**演得出来**：有潜台词、有空白、允许多种读法

```
念得出来的台词（差）：
  "我非常生气，但我不能表现出来，因为我不想让你难过。"

演得出来的台词（好）：
  "（停顿）没事。"
```

**第二种台词**给了演员三个层次：
- 表层：说没事
- 中层：嘴上说没事其实不是
- 底层：不说是因为不想让对方看见

### 1.2 给演员的"钩子"设计

每个有重要戏份的角色都应该有以下"表演钩子"：

**动作钩子**：一个习惯性的身体动作
- 紧张时搓手指
- 思考时咬下唇
- 说谎时下意识看侧面
- 认真时会抿嘴

**物件钩子**：一个始终跟着角色的物件
- 永远戴着的戒指
- 口袋里的打火机
- 皮夹里的一张老照片

**语言钩子**：一个只有这个角色会说的话
- 口头禅
- 特定的语气词
- 方言残留

**气息钩子**：呼吸的节奏
- 焦虑时短促
- 沉着时深长
- 崩溃时憋住又爆发

这些钩子应该在剧本的人物小传部分就写明，拍摄时导演和演员会反复回到这里。

### 1.3 台词的"空白设计"

好的台词必须留出空白让演员填满：

**技法 A：省略号**
```
"你那天……" / "我本来想……"
（给演员演"想说又不说"的空间）
```

**技法 B：问而不答**
```
A："为什么？"
B：（看着 A，没有回答）
```

**技法 C：话题跳转**
```
"（深呼吸）今天你妈打电话了。"
"（另一个话题）电视修好了吗？"
```

**技法 D：重复对方的话**
```
A："我要离婚。"
B："离婚。"
（仅仅是重复，但演员可以演震惊/讽刺/绝望/接受）
```

# craft-layers · §2 动作描写(给摄影留空间)
## 二、动作描写：给摄影留空间

### 2.1 摄影意图 vs 摄影指令

**摄影指令（差）**：
```
镜头从老王的脸上切出，推到手表上，然后拉远到窗外。
```

**摄影意图（好）**：
```
老王看了一眼手表。
窗外，天已经黑了。
```

第二种写法让摄影师自己决定怎么拍，但叙事的节奏（从手表到窗外）是清晰的。

### 2.2 关键物件的"导演指令暗写"

如果你确实需要某个物件被特写，不要直接写"特写"，而是用"行为语言"暗示：

```
【暗示特写的写法】
她把信封在手里转了转。烧焦的边缘贴着她的指尖。

【暗示长镜头的写法】
他一路走到阳台。阳台上还摆着那盆花。花开了。

【暗示声音的写法】
房间突然安静下来。只有冰箱在嗡嗡响。
```

专业的导演和摄影会从这些描述中自动提取机位和剪辑意图。

### 2.3 动作的"分层"

剧本中的动作描写应该有层次：

**第一层：主要动作（必须写）**
"他关上门。"

**第二层：修饰动作（可写可不写）**
"他关上门。回头看了一眼空荡荡的办公室。"

**第三层：心理化动作（谨慎使用）**
"他关上门。回头看了一眼空荡荡的办公室。手指无意识地摩挲着门把手。"

层次越多，表演空间越丰富，但也限制了演员的发挥。一般而言：
- 关键戏（情感高潮）：写到第三层
- 过渡戏：写第一层就够
- 对白戏：只有对白前后有动作时才写

# craft-layers · §7 角色表演档案
## 七、角色表演档案：给演员的"角色辞典"

### 7.1 档案内容

```
【角色：XXX】

外貌 / 体态 / 年龄 / 职业：
  [基本信息]

Ghost / Lie / Flaw：
  [深层设计]

Want / Need：
  [外在+内在欲望]

关键关系：
  与 [角色 A]：[关系状态和历史]
  与 [角色 B]：...

语言指纹：
  高频词汇：
  口头禅：
  禁用词汇：
  句式节奏：

表演钩子：
  动作钩子：
  物件钩子：
  语言钩子：
  气息钩子：

角色的"秘密"：
  [观众不一定知道，但演员必须知道的]

角色的"身体记忆"：
  [角色小时候或过去的身体经历，影响现在的身体语言]
```

### 7.2 秘密设定的价值

**演员需要知道观众不知道的事。**
如果角色在演一个"表面快乐内心崩溃"的戏，演员必须知道"内心为什么崩溃"——哪怕这个原因永远不会被观众知道。

示例：
- 观众知道：这个父亲最近情绪不稳定
- 演员知道：这个父亲在三个月前被诊断出癌症，但没告诉任何人

演员演的时候会不自觉地带入"癌症诊断"这个知识，观众虽然不知道原因但能感受到"不对劲"——这就是层次感。

# AI 腔对策 · §1-4 心理/比喻/解释/词汇
## 一、心理描写→动作替换

AI 最容易出现的错误。观众无法看到角色"内心"，只能通过行为推断。

| ❌ AI 腔（心理描写） | ✅ 人类写法（动作替换） |
|--------------------|----------------------|
| 他感到一阵心痛。 | 他把烟按灭在纸条上。 |
| 她意识到自己爱上了他。 | 她开始记他每天穿什么衣服。 |
| 他内心涌起一股暖意。 | 他把自己的外套脱下来，盖在孩子身上。 |
| 她陷入了深深的绝望。 | 她坐在地板上，鞋都没脱。 |
| 他回忆起童年的那个下午。 | 他看到院子里的秋千，脚步慢了半拍。 |
| 她突然明白了一切。 | 她把手里的信揉成一团，又慢慢展开。 |
| 他心里翻江倒海。 | 他喝了一口水，杯子放下时比端起时轻了一点。 |
| 她对他产生了怀疑。 | 她递过去的钥匙，还没放下就收了回来。 |

---

## 二、书面化比喻→口语化

AI 喜欢"文学感"，真人对话没有这么多比喻。

| ❌ AI 腔（过度比喻） | ✅ 人类写法（朴素直白） |
|--------------------|----------------------|
| 时间像沙漏一样从指间流逝。 | 都快十点了。 |
| 她的眼泪如断了线的珍珠。 | 她擦了擦脸。 |
| 命运的齿轮开始转动。 | （删掉） |
| 这一刻，时间仿佛静止了。 | （用镜头语言代替，不写字） |
| 他的心如同被千万根针扎。 | 他没吭声。 |
| 阳光温柔地洒在她身上。 | 阳光从左边窗户进来。 |
| 沉默像一堵无形的墙。 | 两个人都没说话。 |

---

## 三、解释性台词→潜台词

台词不应该说出双方都已经知道的事，更不应该解释剧情。

| ❌ AI 腔（解释性） | ✅ 人类写法（潜台词） |
|------------------|---------------------|
| "我很害怕失去你。" | "你走的时候把那件外套带上。外面冷。" |
| "其实我一直爱着你。" | "你那件衬衫……我还留着。" |
| "我知道你在骗我。" | （看着他，没说话） |
| "我们不适合在一起。" | "我明天去南京。" |
| "我对你失望了。" | "你吃饭了吗？"（明知道吃了） |
| "这一切都是我的错。" | "对不起。"（简单到过分） |
| "你不懂我。" | "嗯。" |

# AI 腔 · §7 对白'不完美'检查
## 七、对白的"不完美"检查

**AI 写的对白太完美。** 真人说话有：

- **假起头**："那个、那个……你吃饭了吗？"
- **打断**："我昨天——算了不说了。"
- **说半句**："所以我想……（换话题）"
- **跑题**："你这伞哪买的？（根本不关心答案）"
- **重复**："你真的？你真的吗？"
- **语气词**：啊、嗯、哎、这个、那个
- **口头禅**：每个角色不同的口头禅
- **方言残留**：带地域色彩的词

**每场戏至少要有 1 句不完美的对白。**

# AI 腔 · §8 场景结尾重复模式
## 八、场景结尾的重复模式

AI 喜欢用相同的方式结束每个场景：
- 角色转身离开
- 镜头拉远到空镜
- 叹一口气
- 手落下来
- 眼泪掉下来

**每个场景的结尾必须独特。**

替代方案：
- 停在一个动作的中途
- 切到完全不相干的画面
- 声音先于画面结束
- 一个新角色突然出现
- 一个物件被故意遗忘

# AI 腔 · §10 五项机械测试
## 十、五项机械测试（每次写完一场戏做一遍）

1. **具体性测试**：是否有具体名词和精确动词，而非抽象形容词堆砌？
   - ❌ "她感到深深的伤心。"
   - ✅ "她把茶倒进水槽里。杯子里还留了一口。"

2. **僵尸测试**：删除所有形容词后，句子是否仍能传达意义？
   - ❌ "她用哭泣的、颤抖的、破碎的声音说……"
   - ✅ "她说……"

3. **气味测试**：大声朗读，听起来像人在说话吗？
   - 如果读起来有朗诵感，就是 AI 腔。

4. **惊喜测试**：有没有一个让人意外的句子？
   - 人类写作有惊喜，AI 永远没有。

5. **替换测试**：把角色名换成别人，台词还成立吗？
   - 成立 = 角色区分不够，需要重写台词。

# AI 腔 · §11 中文剧本特有 AI 腔警告
## 十一、中文剧本特有的 AI 腔警告

### 11.1 书面化的"四字词"

AI 特别喜欢用四字词拼贴：
- ❌ "百感交集 / 感慨万千 / 五味杂陈 / 百转千回"
- ✅ 替换为具体场景（用画面表达）

### 11.2 过度书面化的抒情

- ❌ "她的眼神里，藏着千言万语。"
- ✅ "她看着他。他低头看着自己的鞋。"

### 11.3 网文套话

- "俊美的脸"、"深邃的眼眸"、"清冷的气质"、"邪魅一笑"
- 这些都是网文套话，真正的剧本描写应该"能被摄影机拍到"
- ❌ "他有着俊美的脸和深邃的眼眸。"
- ✅ "他比照片上老。右眼角有一道疤。"

### 11.4 过度戏剧化的反应

- ❌ "她的瞳孔猛地一缩 / 她倒吸一口凉气 / 他的手不受控制地颤抖"
- ✅ 用更微小的动作："她停了一下 / 他把手插进口袋"

# 开场钩子 · 通用公式 (§1 六选一)
## 一、通用开场钩子公式

**公式 A：结果先行**
> 一个离奇的画面 → 全片回答"这是怎么发生的"
> 示例：人在雪地里倒下（《冰血暴》）

**公式 B：规则建立 + 破规则**
> 先展示规则 → 立即展示破规则的代价
> 示例：不能发声的世界，孩子按了玩具（《寂静之地》）

**公式 C：日常裂缝**
> 完美的日常 → 一个微小不对劲的细节
> 示例：是枝裕和开场的家庭早餐，镜头多停了一秒

**公式 D：动作中的角色**
> 角色在做一件日常任务，但任务本身展示其能力
> 示例：丛林中的印第安纳·琼斯（《夺宝奇兵》）

**公式 E：反派先行**
> 主角还没出场，反派先建立
> 示例：银行劫案（《黑暗骑士》）

**公式 F：谜题开场**
> 一段不可理解的对白或画面
> 示例：《记忆碎片》的倒放照片

# 开场钩子禁忌清单 (§4 12 条)
## 四、开场钩子的"禁忌清单"

- ❌ **旁白交代背景**：用画面展示，不用嘴说
- ❌ **闪回开场**：除非有明确的结构理由（如《记忆碎片》）
- ❌ **梦境开场**：用烂的套路
- ❌ **闹钟响，主角醒来**：最平庸的开场
- ❌ **镜头缓慢推入城市**：商业片不要这样开场
- ❌ **角色自我介绍**：非常尴尬
- ❌ **logo 慢入+标题淡出**：观众已经滑走了
- ❌ **日记旁白流水账**：观众没耐心
- ❌ **以天气/风景开场**（除非是美学明确的文艺片）
- ❌ **父母吵架 + 孩子逃出家**：用烂
- ❌ **晨跑主角**：用烂
- ❌ **咖啡厅点餐**：用烂

## 最高优先级铁律 (覆盖所有其他规则)

**这是剧本, 不是小说.**

- 每一行要么是**可被摄影机拍到的动作/画面**, 要么是**可被演员说出的对白**
- 禁止任何观众在成片里"看不见"的内容 (心理描写 / 暗示 / 意识流 / 叙述者视角氛围铺陈)
- 禁止散文式长段落流动句 (用 — 叠三层的那种)
- 对白**必须**以 `角色名：` 前缀独立成行, 禁止用引号 "..." 埋进叙述段

---

## 剧本格式硬约束 (严格按 SKILL core §5)

### 每场 body 的行级结构

body 必须由**两种行类型交替**组成, 段落之间用 \n\n 分隔:

**行类型 A · 动作/画面描写行**
- 1-3 个短句, 每句 5-25 字
- 纯视觉 (摄影机能拍到) + 声音 (麦克风能录到)
- 可以有情境信息 (时间 / 环境变化 / 物件特写暗示)

**行类型 B · 对白行**
- 严格前缀: `角色名：对白内容。`
- 可选语气提示: `角色名（低声）：` / `角色名（哽咽）：` / `角色名（O.S.）：` / `角色名（V.O.）：`
- 禁止: 引号 "..." 包住对白埋在叙述段里

### 示例结构
```
动作行。

动作行。

角色名：对白。

角色名：对白。

动作行。
```

### scene.body **不要**重复 scene header 或 节奏元数据

header / duration / plotRhythm / emotionRhythm 是**独立字段**, body 只装剧本正文行.
- ❌ body 开头写 "【场景 1：...】" — header 字段已有
- ❌ body 开头写 "情节节奏：紧 | 情感节奏：重" — 节奏字段已有

## 最后 1 场 (act4 新常态) 写作硬约束

最后 1 场 (act="act4" 的 scene) 必须按 **core §2.7 line 149 "回调初遇场景, 展示走过的距离"** 技法写:

### 必做 · 回调对照

- 找 act1 建置里的某个具体元素 (场景 / 符号物件 / 对白 / 关系状态 / 一个习惯动作)
- 在新常态场里, 让这个元素以"变化后"的形态再现

示例 · 红绳回调:

- act4 · 新常态 (回调): 阿阮自己重新系红绳, 陈陌在旁边等 → 象征"伙伴关系, 我陪你"

### body 结构

不是象征空镜合集, 是**角色变化后怎么生活**的片段:
- 具体动作 (不是纯观察 / 纯走路)
- 具体互动 (关系的新状态通过动作展示)
- 具体世界细节 (不是抽象远方)
- 30-60 秒上限 (format-short 陷阱 line 287)

### 禁忌 (0 命中)

- ❌ 纯象征空镜合集 (晨光/夕阳/乌云散/走向远方/背影融入风景)
- ❌ 纯抽象 ("成为朋友开始新生活")
- ❌ 站着说教 / 旁白总结
- ❌ 超 60 秒长尾拖尾
- ❌ 0 回调 act1 (新常态和建置没任何呼应 = fail)

## 按场次类型差异化写作 (craft-layers §2.3 分层原则)

读 Step 6 给出的 `plotRhythm` × `emotionRhythm` 判断场次类型, 选对应写法:

| plotRhythm × emotionRhythm | 类型 | 写法 |
|---|---|---|
| 紧 × 重 | **关键戏** (情感高潮 / 冲突顶点) | 对白密集 + 动作短句爆发 · 可用 craft-layers §2.3 三层 (但仍是"动作行 + 对白行交替", 不是叠散文) |
| 紧 × 轻 | **动作戏** (追逐 / 打斗) | 连续动作行, 每行 5-15 字, 快剪感, 对白极少 |
| 松 × 重 | **情感戏** (分离 / 独处 / 告白) | 对白稀 + 动作极简 + 留白多 (段间多空行) |
| 松 × 轻 | **过渡戏** (场景连接 / 空间建立) | 1-3 段动作描写, 通常无对白 |

**关键**: craft-layers §2.3 原文 — *"分层越多, 表演空间越丰富, 但也限制了演员的发挥"*. 不是每场每段都三层叠一起, 按场次类型选深度.

## 对白写法 (本片核心质量指标)

### 引用 SKILL 章节
- core §3.2 潜台词 (Subtext) — 对话像冰山
- core §3.4 台词风格 — 口语化
- craft-layers §1.1 "演得出来" vs "念得出来"的台词
- craft-layers §1.3 台词空白设计 (省略号 / 问而不答 / 话题跳转 / 重复对方)
- ai-pitfalls §3 解释性台词 → 潜台词对照表
- ai-pitfalls §7 对白"不完美"检查

### 硬规则

1. **对白占比**: body 字数的 40-60% (本片核心质量)
2. **潜台词承载**: 至少 1/3 对白**不是字面意思** (说 A 指 B)
3. **语言指纹区分**: 读 Step 3 characters[].linguistics 的 freqWords / catchphrase / gesture / speechRhythm, 每个角色的对白**必须通过剥标签测试** (去掉角色名能否辨认)
4. **每场至少 1 句不完美对白** (ai-pitfalls §7): 假起头 / 打断 / 说半句 / 重复 / 语气词 / 方言残留

### 台词空白设计 · 4 技法 (按 craft-layers §1.3 原文)

好的台词必须留出空白让演员填满. 至少用以下 4 技法之一:

**技法 A · 省略号**: 给演员演"想说又不说"的空间
```
陆沉舟：你那天……
（对方不追问, 两人继续做手头的事）
```

**技法 B · 问而不答**: 用沉默代替答案
```
阿阮：为什么？
陆沉舟：（看着阿阮, 没有回答。转身倒水。）
```

**技法 C · 话题跳转**: 情感太重时用另一个话题转移
```
母亲：（深呼吸）今天你妈打电话了。
儿子：电视修好了吗？
```

**技法 D · 重复对方的话**: 仅仅重复, 但演员可以演震惊/讽刺/绝望/接受
```
妻子：我要离婚。
丈夫：离婚。
```

**对照 §1.1 "演得出来" vs "念得出来"** (原文):
- ❌ 念得出来 (差): "我非常生气, 但我不能表现出来, 因为我不想让你难过。"
- ✅ 演得出来 (好): "（停顿）没事。"

### 禁用对白 (ai-pitfalls §3 原文对照)

```
❌ "我很害怕失去你"       → ✅ "你走的时候把那件外套带上。外面冷。"
❌ "其实我一直爱着你"       → ✅ "你那件衬衫……我还留着。"
❌ "我知道你在骗我"         → ✅ (看着他, 没说话)
❌ "我对你失望了"           → ✅ "你吃饭了吗？" (明知道吃了)
❌ "这一切都是我的错"       → ✅ "对不起。" (简单到过分)
❌ "你不懂我"               → ✅ "嗯。"
```

### 引用 SKILL 章节
- core §3.1 Show, Don't Tell
- core §3.3 动作即潜台词
- craft-layers §2.1 摄影意图 vs 摄影指令
- craft-layers §2.2 关键物件的导演指令暗写

### 硬规则

   - ❌ "瞳孔收缩" / "指节发白" / "嘴角抽搐" / "不由自主地颤抖" (心理暗示伪装成动作)

## scene 字段定义

### header 字段 · **必须单行**

✅ 正确: `"header": "【场景 1：破庙 / 雨夜 / 炭盆跳火光】"`
❌ 错误: `"header": "【场景 1：...】（约 45 秒）\n  情节节奏：紧 | 情感节奏：重"`

**硬规则**:
- 纯标题字符串, 单行
- 不含换行符 \n
- 不含时长括号（约 X 秒） — duration 是独立字段
- 不含节奏元数据 — plotRhythm / emotionRhythm 是独立字段

### duration / plotRhythm / emotionRhythm 字段

继承 Step 6 对应字段. **不修改. 用于客户端渲染.**

### duration 字符串硬约束

- **精确继承 Step 6 durationSec** · 严禁舍入/压缩 · 写法: 45 → "约 45 秒" / 60 → "约 1 分钟" / 90 → "约 1 分 30 秒"
- **sum(scene.duration 解析秒数) ≥ Step 6 总和 × 0.95** = ≥ 用户 init 总时长 × 0.95
- 短于 95% = 产品失败 · 优先级高于任何其他 Step 7 写作要求

### body 字段 · 剧本正文

- 按上述行级格式硬约束写
- 段落用 \n\n 分隔
- 不含场景头 / 不含节奏元数据

### hookFormula 字段 · 第 1 场必填 (scenes[0]), 其他场留空

- 枚举: "A" | "B" | "C" | "D" | "E" | "F" (对应 genre-hook-library §1 六选一)
- **只有 scenes[0] 必填**, 其他场无需此字段
- 客户端不消费此字段, 纯供 LLM 自我声明 + Step 8 医生检查
