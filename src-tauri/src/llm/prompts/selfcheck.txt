# 核心方法论 · §4 自检体系（五大板块）
## 四、自检体系

每个工作流步骤都有对应的自检清单。用户可以在任何步骤要求执行自检。

### 自检一：核心概念与戏剧动作
- [ ] 是否建立了"目标+冲突"？
- [ ] 目标是否合理且有急切性？
- [ ] 目标和阻碍是否有强烈关系？
- [ ] 冲突是否是驱动故事的发动机？
- [ ] 阻碍是否在合理尺度内，而非刻意夸大？

### 自检二：人物塑造与弧光
- [ ] 外在需求和内在欲望是否清晰且对立？
- [ ] 是否预设了不可逆的内在变化？
- [ ] 结局是否展示了"新常态"？
- [ ] 是否具备矛盾性？
- [ ] 行为是否符合小传逻辑？

### 自检三：结构与节奏
- [ ] 结构各部分是否清晰？
- [ ] 激励事件是否明确？
- [ ] 高潮是否明确？
- [ ] 每一部分是否都在推动情节或揭示性格？
- [ ] 是否存在无效情节？

### 自检四：锻字（场景写作）与视听语言
- [ ] 是否有心理描写？（如有则Fail）
- [ ] 是否有括号暗示？（如有则Fail）
- [ ] 是否用画面叙事？
- [ ] 对话是否过于直白？
- [ ] 是否利用了潜台词和肢体语言？
- [ ] 台词是否口语化？
- [ ] 是否有角色在"解释设定"而非"应对设定"？
