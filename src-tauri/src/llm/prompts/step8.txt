# AI 腔速查对照表

> 剧本写作中最容易暴露 AI 味的典型表达。写作时随手查，输出前扫一遍。

---

## 一、心理描写→动作替换

AI 最容易出现的错误。观众无法看到角色"内心"，只能通过行为推断。

| ❌ AI 腔（心理描写） | ✅ 人类写法（动作替换） |
|--------------------|----------------------|
| 他感到一阵心痛。 | 他把烟按灭在纸条上。 |
| 她意识到自己爱上了他。 | 她开始记他每天穿什么衣服。 |
| 他内心涌起一股暖意。 | 他把自己的外套脱下来，盖在孩子身上。 |
| 她陷入了深深的绝望。 | 她坐在地板上，鞋都没脱。 |
| 他回忆起童年的那个下午。 | 他看到院子里的秋千，脚步慢了半拍。 |
| 她突然明白了一切。 | 她把手里的信揉成一团，又慢慢展开。 |
| 他心里翻江倒海。 | 他喝了一口水，杯子放下时比端起时轻了一点。 |
| 她对他产生了怀疑。 | 她递过去的钥匙，还没放下就收了回来。 |

---

## 二、书面化比喻→口语化

AI 喜欢"文学感"，真人对话没有这么多比喻。

| ❌ AI 腔（过度比喻） | ✅ 人类写法（朴素直白） |
|--------------------|----------------------|
| 时间像沙漏一样从指间流逝。 | 都快十点了。 |
| 她的眼泪如断了线的珍珠。 | 她擦了擦脸。 |
| 命运的齿轮开始转动。 | （删掉） |
| 这一刻，时间仿佛静止了。 | （用镜头语言代替，不写字） |
| 他的心如同被千万根针扎。 | 他没吭声。 |
| 阳光温柔地洒在她身上。 | 阳光从左边窗户进来。 |
| 沉默像一堵无形的墙。 | 两个人都没说话。 |

---

## 三、解释性台词→潜台词

台词不应该说出双方都已经知道的事，更不应该解释剧情。

| ❌ AI 腔（解释性） | ✅ 人类写法（潜台词） |
|------------------|---------------------|
| "我很害怕失去你。" | "你走的时候把那件外套带上。外面冷。" |
| "其实我一直爱着你。" | "你那件衬衫……我还留着。" |
| "我知道你在骗我。" | （看着他，没说话） |
| "我们不适合在一起。" | "我明天去南京。" |
| "我对你失望了。" | "你吃饭了吗？"（明知道吃了） |
| "这一切都是我的错。" | "对不起。"（简单到过分） |
| "你不懂我。" | "嗯。" |

---

## 四、AI 高频词汇（见即删除）

```

encompass, multifaceted, tapestry, testament, paradigm, synergy,

```

### 中文 AI 腔
```

心智、场域、生态位、破圈、出圈、共情、沉浸式、颗粒度
```

### AI 作文模板短语
```

In other words / Ultimately / At the end of the day /
首先…其次…最后、一方面…另一方面、不仅…而且
```

**一段中出现 3 个以上就需要整段改写。**

## 五、AI 结构陷阱

| 特征 | 人话改法 |
|------|---------|
| 固定段落模板（主题句→展开→举例→总结） | 打乱顺序，从中间开始 |
| 段落长度可疑地均匀 | 制造长短对比 |
| 连续两段以上以转折词开头（然而/此外） | 从主语开始 |
| 每段破折号超过 1 个 | 减到 0-1 个 |
| 句子长度缺乏变化 | 人类标准差 >4.7 词，要长短混搭 |
| "三项一组"的列举过多（不仅是 A，还是 B，更是 C） | 换成两项或四项 |

---

## 六、AI 稳定性陷阱（情感层）

AI 写作倾向于"让一切和解"，人类故事需要"留下伤痕"。

| ❌ AI 默认倾向 | ✅ 人类故事需要 |
|--------------|---------------|
| 角色结尾与开头相似 | 角色必须真正改变 |
| 一切伤害都可修复 | 必须有不可逆的损失或决定 |
| 信息立即揭示 | 故意隐瞒部分真相 |
| 情感强度保持中等 | 安静→爆发→恐惧→释放→麻木 的起伏 |
| 选择都无痛 | 必须有真正的代价 |
| 反派最终被���服 | 反派可以不被说服 |
| 冲突都被解决 | 可以有"不解决"的结尾 |
| 主角一直是对的 | 主角可以有真正的错误 |

---

## 七、对白的"不完美"检查

**AI 写的对白太完美。** 真人说话有：

- **假起头**："那个、那个……你吃饭了吗？"
- **打断**："我昨天——算了不说了。"
- **说半句**："所以我想……（换话题）"
- **跑题**："你这伞哪买的？（根本不关心答案）"
- **重复**："你真的？你真的吗？"
- **语气词**：啊、嗯、哎、这个、那个
- **口头禅**：每个角色不同的口头禅
- **方言残留**：带地域色彩的词

**每场戏至少要有 1 句不完美的对白。**

---

## 八、场景结尾的重复模式

AI 喜欢用相同的方式结束每个场景：
- 角色转身离开
- 镜头拉远到空镜
- 叹一口气
- 手落下来
- 眼泪掉下来

**每个场景的结尾必须独特。**

替代方案：
- 停在一个动作的中途
- 切到完全不相干的画面
- 声音先于画面结束
- 一个新角色突然出现
- 一个物件被故意遗忘

---

## 九、比喻来源的一致性

**比喻必须来自角色的经历领域。**

- 铁匠用热和金属思考：" 像刚淬火的铁。"
- 水手用海和风思考："像平静的死海。"
- 医生用病理思考："像转移的癌细胞。"
- 农民用天气思考："像要下雨了。"
- 程序员用代码思考："像无限循环。"

❌ 错误：一个农民用"像转移的癌细胞"这种比喻——脱离角色认知范围。

---

## 十、五项机械测试（每次写完一场戏做一遍）

1. **具体性测试**：是否有具体名词和精确动词，而非抽象形容词堆砌？
   - ❌ "她感到深深的伤心。"
   - ✅ "她把茶倒进水槽里。杯子里还留了一口。"

2. **僵尸测试**：删除所有形容词后，句子是否仍能传达意义？
   - ❌ "她用哭泣的、颤抖的、破碎的声音说……"
   - ✅ "她说……"

3. **气味测试**：大声朗读，听起来像人在说话吗？
   - 如果读起来有朗诵感，就是 AI 腔。

4. **惊喜测试**：有没有一个让人意外的句子？
   - 人类写作有惊喜，AI 永远没有。

5. **替换测试**：把角色名换成别人，台词还成立吗？
   - 成立 = 角色区分不够，需要重写台词。

---

## 十一、中文剧本特有的 AI 腔警告

### 11.1 书面化的"四字词"

AI 特别喜欢用四字词拼贴：
- ❌ "百感交集 / 感慨万千 / 五味杂陈 / 百转千回"
- ✅ 替换为具体场景（用画面表达）

### 11.2 过度书面化的抒情

- ❌ "她的眼神里，藏着千言万语。"
- ✅ "她看着他。他低头看着自己的鞋。"

### 11.3 网文套话

- "俊美的脸"、"深邃的眼眸"、"清冷的气质"、"邪魅一笑"
- 这些都是网文套话，真正的剧本描写应该"能被摄影机拍到"
- ❌ "他有着俊美的脸和深邃的眼眸。"
- ✅ "他比照片上老。右眼角有一道疤。"

### 11.4 过度戏剧化的反应

- ❌ "她的瞳孔猛地一缩 / 她倒吸一口凉气 / 他的手不受控制地颤抖"
- ✅ 用更微小的动作："她停了一下 / 他把手插进口袋"

## 十二、剧本完稿前最终扫描

按此清单对整个剧本做最后一遍扫描：

- [ ] 所有"他觉得/她意识到/心中想"全部删除或改写成动作
- [ ] 所有括号里的情绪暗示"（伤心地）""（愤怒地）"全部删除
- [ ] 所有 AI 高频词汇（第四节清单）全部替换
- [ ] 每个场景结尾检查一遍——有没有重复模式？
- [ ] 每个角色的对白遮住名字——能不能分辨是谁？
- [ ] 朗读全剧——有没有朗诵感的段落？
- [ ] 把对白删掉一半——还能看懂吗？（如果看不懂，说明过度依赖对白）
- [ ] 有没有一段话里 3 个以上的四字词？
- [ ] 有没有一个让人意外的句子？（如果没有，是 AI 感）
- [ ] 情感强度是否有起伏？（不是平线）
- [ ] 是否有不可逆的损失或决定？

通过这个清单 = 剧本摆脱 AI 腔。


# 开场钩子禁忌清单 (§4)
## 四、开场钩子的"禁忌清单"

- ❌ **旁白交代背景**：用画面展示，不用嘴说
- ❌ **闪回开场**：除非有明确的结构理由（如《记忆碎片》）
- ❌ **梦境开场**：用烂的套路
- ❌ **闹钟响，主角醒来**：最平庸的开场
- ❌ **镜头缓慢推入城市**：商业片不要这样开场
- ❌ **角色自我介绍**：非常尴尬
- ❌ **logo 慢入+标题淡出**：观众已经滑走了
- ❌ **日记旁白流水账**：观众没耐心
- ❌ **以天气/风景开场**（除非是美学明确的文艺片）
- ❌ **父母吵架 + 孩子逃出家**：用烂
- ❌ **晨跑主角**：用烂
- ❌ **咖啡厅点餐**：用烂

# 开场钩子五项自检 (§5)
## 五、开场钩子的五项自检

用于第五步结构大纲完成后：

1. **前 X 秒的内容（按格式）是否有视觉冲击或悬念？**
2. **观众是否会产生具体疑问？**（不是"他是谁"而是"为什么他要做这个？"）
3. **开场是否"在制造张力的同时交代状态"？**
4. **开场的类型基调是否与全片一致？**
5. **如果把开场单独拿出来，它是否是一段精彩的视听内容？**

五项全过 = 开场钩子过关。
