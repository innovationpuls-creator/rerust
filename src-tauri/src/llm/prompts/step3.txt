# 核心方法论 · §2 人物设计 全章
## 二、人物设计：拒绝纸片人

### Want vs Need

- **外在需求 (Want)**：驱动动作的目标，角色自己意识到的、主动追求的东西
- **内在欲望 (Need)**：角色自己未曾意识到的灵魂缺口，通常是外在欲望的深层原因

Want和Need之间的张力是角色弧光的发动机。角色在故事开始时追求Want，在故事结束时要么得到了Need（成长弧光），要么拒绝了Need（悲剧弧光/反弧光）。

### 人物弧光

弧光 = 从A状态到B状态的不可逆变化。
短内容的弧光是一次性的跳跃（A→B）。
长内容的弧光可能经历多次"伪变化"——角色看起来变了，但又退回去，直到最终的真正转变。

### 人物小传

角色的前史不是背景介绍，而是冲突的根源。小传中的关键事件应该解释：
- 为什么角色有这个Want
- 为什么角色看不到自己的Need
- 角色的矛盾性从何而来

### 矛盾性

好的角色至少有一层矛盾：外在表现 vs 内在真实。
例如：外在自信 vs 内在自卑，外在冷漠 vs 内在渴望连接。
矛盾性让角色立体，也让潜台词有了空间。

### 人物深度缩放

| 格式 | 人物深度要求 |
|------|------------|
| 概念超短片 | 功能性archetype即可，不需要小传和弧光 |
| 短片 | Want/Need/Arc + 关键矛盾特征 |
| 长片 | 完整小传 + Ghost(前史创伤) + Lie(角色相信的谎言) + Flaw(性格缺陷) |
| 剧集 | 同长片深度，但弧光需分配到各集，另需设计"弧光预算" |

# craft-layers · §1.2 给演员的钩子设计
### 1.2 给演员的"钩子"设计

每个有重要戏份的角色都应该有以下"表演钩子"：

**动作钩子**：一个习惯性的身体动作
- 紧张时搓手指
- 思考时咬下唇
- 说谎时下意识看侧面
- 认真时会抿嘴

**物件钩子**：一个始终跟着角色的物件
- 永远戴着的戒指
- 口袋里的打火机
- 皮夹里的一张老照片

**语言钩子**：一个只有这个角色会说的话
- 口头禅
- 特定的语气词
- 方言残留

**气息钩子**：呼吸的节奏
- 焦虑时短促
- 沉着时深长
- 崩溃时憋住又爆发

这些钩子应该在剧本的人物小传部分就写明，拍摄时导演和演员会反复回到这里。

# craft-layers · §7 角色表演档案
## 七、角色表演档案：给演员的"角色辞典"

### 7.1 档案内容

```
【角色：XXX】

外貌 / 体态 / 年龄 / 职业：
  [基本信息]

Ghost / Lie / Flaw：
  [深层设计]

Want / Need：
  [外在+内在欲望]

关键关系：
  与 [角色 A]：[关系状态和历史]
  与 [角色 B]：...

语言指纹：
  高频词汇：
  口头禅：
  禁用词汇：
  句式节奏：

表演钩子：
  动作钩子：
  物件钩子：
  语言钩子：
  气息钩子：

角色的"秘密"：
  [观众不一定知道，但演员必须知道的]

角色的"身体记忆"：
  [角色小时候或过去的身体经历，影响现在的身体语言]
```

### 7.2 秘密设定的价值

**演员需要知道观众不知道的事。**
如果角色在演一个"表面快乐内心崩溃"的戏，演员必须知道"内心为什么崩溃"——哪怕这个原因永远不会被观众知道。

示例：
- 观众知道：这个父亲最近情绪不稳定
- 演员知道：这个父亲在三个月前被诊断出癌症，但没告诉任何人

演员演的时候会不自觉地带入"癌症诊断"这个知识，观众虽然不知道原因但能感受到"不对劲"——这就是层次感。

## 语言指纹 · 按 [core §2.5 角色语言指纹] (短片以上格式必做)

每个主要角色必须建立**语言档案**, 通过"剥标签测试":
> 给一段 80-120 字的角色对白, 去掉角色名和提示标记, 读者能否从"高频词汇 + 句式节奏 + 标志动作"判断出是谁? 不能 → 指纹不及格, 重新设计.

每角色按 **core §2.5 词汇画像 + craft-layers §7.1 语言指纹** 原文:
- **freqWords** (高频词汇 3-5 个): 角色真会反复说的实词
- **catchphrase** (口头禅 1-2 个): 标志性短语
- **forbiddenWords** (禁用词汇 2-3 个): 该角色**绝不会说**的词 (用语习惯 / 教育背景 / 性格决定, 例: 学者角色不说"靠谱" / 江湖汉子不说"理论上")
- **gesture** (说话时伴随动作 2-3 个): 说话时的习惯性肢体动作 (与 actingHooks.gesture 可相同可不同 — linguistics.gesture 偏"对话伴随", actingHooks.gesture 偏"角色标志")
- **sampleLines** (代表性台词样本 3 句, 长片/剧集必填 · 短片可选): 最能体现角色特征的 3 句话, 用于 AI 长内容创作中的角色语音维持
- **speechRhythm** (句式节奏, 从以下挑 1 个):
  - "strongShort" 强势短句 (3-6 字短句)
  - "hesitantLong" 犹豫长句 (10-20 字长句)
  - "cunning" 老谋深算 (8-12 字复合句)
  - "impulsive" 冲动感叹 (2-5 字感叹句)

## 给演员的表演钩子 · 按 [craft-layers §1.2 + §7 角色表演档案]

linguistics 是**语言指纹** (观众通过对白辨认角色). 表演钩子是**肢体指纹** (观众通过身体语言记住角色). 两套系统不冲突, 可共存.

每角色填 actingHooks (短片以上格式必做):
- **gesture** — 动作钩子 (1 个习惯性身体动作, 如"紧张时搓手指" / "思考时摸下巴")
- **prop** — 物件钩子 (1 个始终跟着的物件, 如"永远戴着的戒指" / "永远拿着的保温杯")
- **language** — 语言钩子 (1 个**只有这角色会说的话**, 区别于 linguistics.catchphrase — 这里是独一无二的说话方式, 如"李火旺说'俺' 不说'我'")
- **breath** — 气息钩子 (呼吸节奏, 如"焦虑时短促" / "紧张时屏住" / "放松时长吐")

另外按 craft-layers §7 加两个字段 (观众不一定看到, 演员必须知道):
- **secret** — 角色的秘密 (观众不一定知道, 演员必须知道的. 如"主角 30 岁生日前夜偷听到父亲要放弃治疗" — 这秘密不会明说, 但会影响他接下来每一场戏的微表情)
- **bodyMemory** — 身体记忆 (过去身体经历如何影响现在的身体语言. 如"三年前车祸后左腿有旧伤, 下雨天走路会无意识左倾")

## 关系工具箱 · 按 [core §2.7 关系设计工具箱] (多角色时必填)

如 characters 数 ≥ 2, 必填 relationships 数组, 描述每对核心关系:

- **dynamicType** (关系动力学类型, 从以下挑):
  - "pushPull" 推拉 (4 规则: 给予后收回 / 亲近后疏离 / ...)
  - "chemistry" 化学反应 (8 技巧之一: 反差配对 / 秘密共享 / ...)
  - "trust" 信任关系 (适用于合伙/师徒)
  - "emotional" 情感线 (适用于爱情/挚友)
- **trustLevel** (0-5 整数, 按 **core §2.7 信任 6 级阶梯原文**: 0 敌意/警戒 / 1 中立容忍 / 2 有限信任 / 3 实质信任 / 4 深层信任 / 5 无条件信任. 背叛冲击方程: 冲击力 = 信任级别 × 不可预期度 × 后果不可逆性, 信任越深背叛越痛)
- **initialState** (故事开始时的关系状态描述)
- **targetState** (故事结束时的预期状态)
- **beats** (仅爱情/挚友线要填, 情感 6 节拍的位置, 如 "初遇 0-15% · 锁定 15-25% · 信物 40-55% · 断裂 65-80% · 豪赌 85-95% · 新常态 95-100%")
