# 核心方法论 · §6 双轨节奏系统
## 六、双轨节奏系统

优秀影片的节奏是张弛有度的。节奏不是只有一条线——它有两条平行运行的轨道，两者可以同步也可以错位，错位本身就是强大的叙事工具。

### 外部情节节奏

外部情节节奏是"事件的密度和强度"——场景中发生了多少事、冲突有多激烈、信息量有多大。

- **紧**：高密度事件、快速剪辑、多人对话、追逐、对峙、揭露
- **松**：日常场景、独处时刻、过渡、风景、沉默

外部节奏的张弛要有对比——持续紧张会让观众麻木，持续松弛会让观众走神。经验法则：每一段高强度情节之后，需要一段"呼吸"让观众消化。

### 内在情感节奏

内在情感节奏是"角色情感状态的波动"——角色经历了多大的情感起伏、观众被要求投入多少情感。

- **重**：告白、背叛、失去、恐惧、狂喜、绝望
- **轻**：调侃、日常闲聊、小幽默、安静的陪伴

情感节奏和情节节奏并不总是同步的。一些最强大的场景恰恰是两者错位的时刻：

- **情节松 + 情感重**：两个人安静地坐着吃饭，什么事都没发生，但观众知道其中一个人明天要走了。画面越平静，情感越重。
- **情节紧 + 情感轻**：激烈的追车戏，但配乐是轻松的流行歌曲，角色在打趣。紧张的事件被轻盈的情绪包裹。
- **情节紧 + 情感重**：高潮段——事件密集且情感炸裂。这是全片的顶点，不能持续太久。
- **情节松 + 情感轻**：纯粹的呼吸场景。让观众休息。通常放在高潮前后。

### 节奏波浪图

在塑骨（结构大纲）和剖镜（场景拆解）阶段，应该为全片画一张"节奏波浪图"——标注每个段落的外部情节强度和内在情感强度，检查是否有足够的起伏和对比。

```
示意（纵轴=强度，横轴=时间线）：

外部情节:  ─╱╲─────╱╲──╱╲╱╲╲──────╱╲╱╲╱╲╲
内在情感:  ──────╱╲───────╱╲──╱╲╱╲───────╲╱╲
```

两条线不必同步，但应该避免：
- 两条线长时间同时处于低位（观众无事可看也无感可投）
- 两条线长时间同时处于高位（观众疲劳，高潮变得不特别）
- 任何一条线单调上升没有回落（没有呼吸感）

# 核心方法论 · §7 时长估算
## 七、时长估算与节奏控制

每一个戏剧动作、每一场戏都应该有时长估算。时长不只是"这场戏多长"，更是"这场戏在全片节奏中占多大比重"。

### 为什么必须估算时长

- 一场戏如果在大纲里读起来很重要，但只占30秒，它在成片里的分量会被压缩到观众几乎注意不到
- 一场戏如果占了全片15%的时长，但只传递了一个次要信息，它就是节奏的拖累
- 只有把时长标出来，才能在纸面上看到节奏的问题——哪里太赶、哪里太拖、高潮是否有足够的铺垫时间

### 时长估算方法

**对白估算：** 1页标准剧本格式 ≈ 1分钟成片。中文剧本中，大约200-250字的对白+动作描写 ≈ 1分钟。

**场景估算规则：**
- 纯对话场景：按对白字数估算
- 动作/蒙太奇场景：按画面数量估算（每个独立画面约3-5秒）
- 沉默/情绪场景：容易低估——一个角色站在那里什么都不做，在成片中可能需要10-15秒才有足够的情感分量
- 转场/过渡：不要忘记计入，通常每次转场2-5秒

### 场景时长标注格式

在剖镜（场景拆解）和锻字（剧本写作）阶段，每场戏都应标注预估时长：

```
【场景3：医院走廊 / 夜 / 荧光灯嗡嗡响】（约1分30秒）
  情节节奏：中（一次关键对话）
  情感节奏：重（角色第一次面对真相）
```

### 全片时长预算

在塑骨（结构大纲）阶段就应该做一份粗略的时长预算，确保各段落的比例合理：

| 格式 | 建置 | 发展前半 | 发展后半 | 高潮 | 结局 |
|------|------|---------|---------|------|------|
| 短片（8分钟） | 1.5分钟 | {duration} | {duration} | 1.5分钟 | 1分钟 |
| 长片（100分钟） | 25分钟 | 30分钟 | 25分钟 | 15分钟 | 5分钟 |
| 剧集单集（25分钟） | 4分钟 | 10分钟 | 6分钟 | 3分钟 | {duration} |

以上为参考比例——实际创作中可以灵活调整，但调整后应该意识到自己在做什么（比如"我故意让建置占了40%时长，因为这是一个慢热型故事"）。

### 节奏诊断

在磨锋（剧本医生）阶段，用时长预算检查以下问题：
- 高潮段的时长是否足够？（太短会让高潮感觉被压缩）
- 建置是否太长？（观众等太久才进入正题）
- 是否有连续超过3分钟的单一情绪段落？（无论是紧张还是松弛，持续太久都会失效）
- 两个高情感场景之间是否有足够的呼吸空间？
- 全片最长的单场戏是否有足够理由占据那么多时长？

# craft-layers · §3 分镜意图标注系统
## 三、分镜意图标注系统

剧本中可以用以下标注传达分镜意图，但不限制导演：

### 3.1 标注格式

```
【场景X：地点 / 时间 / 视觉描述】（约 X 分 X 秒）
  情节节奏：松/中/紧 | 情感节奏：轻/中/重
  【镜头意图】：[可选，给导演参考]
  【美术重点】：[可选，给美术参考]
  【声音层】：[可选，给音效参考]
  【音乐】：[可选，给作曲参考]

……
```

### 3.2 镜头意图分类

**"冷静观察"：**
- 摄影机与角色保持距离
- 中远景为主
- 固定机位或缓慢移动
- 用于日常、文艺、客观审视

**"贴近共情"：**
- 摄影机紧贴角色
- 中景和特写为主
- 手持或稳定器跟随
- 用于亲密、痛苦、主观体验

**"压迫包围"：**
- 摄影机从角色的侧后方或斜上方
- 用环境压过角色
- 广角变形
- 用于恐惧、权力失衡、心理困境

**"炫技呈现"：**
- 摄影机大幅度移动
- 复杂调度
- 特效辅助
- 用于高潮、概念片、视觉奇观

### 3.3 标注示例

```
【场景 7：医院走廊 / 夜 / 荧光灯频闪】（约 1 分 30 秒）
  情节节奏：中 | 情感节奏：重
  【镜头意图】：贴近共情——摄影机始终在主角身后 1 米，主角走多快摄影机走多快
  【美术重点】：走廊尽头的窗户要能看到远处的霓虹灯，跟荧光灯的冷白形成对比
  【声音层】：主角的呼吸压过环境音；结尾处主角停下时环境音突然放大（心电图声、脚步声）
  【音乐】：不起音乐。只在最后 5 秒进一个单音大提琴长音
```

## 双轨节奏硬规则 · 按 [core §6 / format-short §6 自检]

1. **情节节奏 × 情感节奏 至少一处错位** (松+重 或 紧+轻)
2. **避免连续 3 场同一节奏状态** (会失效)
3. **相邻场景必须有情节过渡** (不要硬切)
4. 节奏波浪应呈现: 建置松 → 发展紧 → 高潮最紧 → 结局回松

- 冲突升级 + 憋屈最高点 (中段)
- 关键转折 (60-70%)
- 爆发反击 (85-100%)
